## Датасет hirefy
Датасет собран с применением selenium с сайта hirefy.me 

Каждая строка 1 карточка вакансии в датасете около 10500 строк
### Файлы
- main.ipynb сам парсинг с сайта

- eda.ipynb процесс очистки

- llm_predict.ipynb файл с процессом предсказания категории по названия с применением llm также изначально алгоритм тестировался на hirehi/test_llm.ipynb

- vacancies.csv результат парсинга

- vacancies_cleaned.csv очистка + llm_predict

- predicted.csv предсказания БЯМ
