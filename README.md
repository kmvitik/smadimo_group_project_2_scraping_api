## Что делает эта часть гп

Собирает вакансии из Adzuna API по Великобритании в категории айти, а затем делает базовый EDA и базовые визуализации.

Используются два ноутбука:
- adzuna_api.ipynb
- adzuna_eda.ipynb
## 1) Сбор данных (adzuna_api.ipynb)

1. Подключение к Adzuna API по app_id, app_key, проверка работоспособности апи.
2. Проверка доступных стран, переход на gb.
3. Получение списка категорий через "/jobs/gb/categories".
4. Фильтрация вакансий по `category=it-jobs`.
5. Тестирование дополнительных эндпоинтов:
   - "/jobs/gb/histogram" - распределение вакансий по зарплатам.
   - "/jobs/gb/top_companies" — топ компаний по числу вакансий.
   - "/jobs/gb/history" - динамика средней зарплаты по месяцам.
   - "/jobs/gb/geodata" - распределение вакансий по регионам.
6. Пример более узкого запроса: "what=data engineer, where=London".
7. Основной сбор датасета:
   - цикл по страницам search/{page} (1..199),
   - логирование прогресса,
   - сбор в DataFrame и сохранение в csv.

Итог после сбора: около **9950 строк** и **17 колонок**.

## 2) Очистка и EDA (adzuna_eda.ipynb)

1. Загрузка adzuna.csv.
2. Удаление технических колонок:
   - Unnamed: 0,
   - adref,
   - __CLASS__,
   - category.
3. Преобразование полей location и company:
   - преобразование строк со словарями в отдельные словари через библу ast,
   - извлечение display_name 
4. Создание признака salary_avg = (salary_min + salary_max) / 2.
5. Построение графиков:
   - топ должностей по salary_avg,
   - распределение salary_avg,
   - топ компаний по средней зарплате,
   - boxplot по salary_avg

## Результат

На выходе:
- собранный датасет вакансий IT по UK из Adzuna,
- базовые графики и статистика по рынку айти в Великобритании.
